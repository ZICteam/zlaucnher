# zlauncher

Java launcher for the local Forge client in this repository.

Current version: `2.0.4`

Features:
- Ely.by authentication
- Ely.by skins support through authlib-injector
- Launches the local Forge profile `1.20.1-forge-47.4.18`
- Downloads missing vanilla metadata, libraries, and assets on first run

Run:

```bash
./launcher/run-launcher.sh
```
